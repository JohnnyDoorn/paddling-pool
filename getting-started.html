Getting Started with JASP

JASP is free, open-source statistical software developed at the University of Amsterdam. It is designed to make statistical analyses intuitive and interactive — output updates in real time as you change settings.

Download and install JASP

Go to jasp-stats.org/download and download the installer for your operating system (Windows, macOS, or Linux). Installation is straightforward; no licence is required.

TipWhich version?

Always download the latest stable release. If you are working on a university computer that already has JASP installed, make sure it is version 0.18 or later — some modules used in this course require a recent version.

Loading data into JASP

JASP can open several file formats: .jasp (JASP’s own format), .csv, .sav (SPSS), .xlsx, .ods, and more.

Opening a .jasp file — this is the easiest way to practise. A .jasp file contains both the dataset and any pre-configured analyses. Double-click the file or use File → Open in JASP.

Opening a .csv or .sav file — use File → Open and navigate to your file. After opening, JASP shows the spreadsheet. Check that your variables have the correct measurement level (nominal, ordinal, or scale) by clicking the variable type icon next to the column header.

The JASP interface

When you open a dataset, the JASP window is divided into three panels:

Panel What it shows
Left — Data The spreadsheet with your variables and cases
Middle — Analysis menu The list of available statistical tests
Right — Output Results that update in real time as you configure the analysis

Variable types

JASP uses three measurement levels:

  • Nominal — unordered categories (e.g., treatment group, gender)
  • Ordinal — ordered categories (e.g., Likert scales treated as ordinal)
  • Scale (continuous/interval/ratio) — numeric measurements (e.g., reaction time, score)

Assigning the correct variable type matters because JASP uses it to determine which analyses are available and how to display your data.

Running an analysis

  1. Select the relevant menu item (e.g., T-Tests → Independent Samples T-Test)
  2. Drag variables into the appropriate boxes (dependent variable, grouping variable)
  3. Tick the options you need (effect size, assumption checks, plots)
  4. Output appears immediately in the right panel

There is no “Run” button — JASP updates results live.

Saving your work

  • File → Save saves a .jasp file containing both data and current analyses
  • To export the output, click the downward triangle (▼) next to any result table or figure and choose Copy, Save as image, or similar
