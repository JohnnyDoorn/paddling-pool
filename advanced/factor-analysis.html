Factor Analysis

Modified

11 June 2026

Advanced Topics

Factor analysis is a family of techniques for identifying latent dimensions (factors) that underlie a set of observed variables. It is widely used in scale development, construct validation, and data reduction.

Three related techniques are covered here:

Technique Purpose
EFA — Exploratory Factor Analysis Discover the underlying factor structure when it is not known in advance
CFA — Confirmatory Factor Analysis Test whether data fit a theoretically specified factor structure
PCA — Principal Component Analysis Data reduction: extract linear combinations that explain maximum variance
WarningEFA ≠ PCA

Although often confused, EFA and PCA have different goals. PCA is a mathematical data reduction technique with no model of latent variables. EFA assumes that observed variables are caused by underlying latent factors plus unique error. Use EFA when you are interested in the constructs; use PCA when you want to reduce dimensionality for practical reasons.

Exploratory Factor Analysis (EFA)

When to use it

  • You have a set of items (e.g., questionnaire items) and you do not know — or want to discover — how many factors underlie them
  • Scale development: determining the dimensional structure of a new measure
  • Generating hypotheses about construct structure to be tested later with CFA

Key decisions

How many factors?

  • Scree plot: look for the “elbow” — the point where the eigenvalues flatten out
  • Kaiser criterion: retain factors with eigenvalue > 1 (can over-extract in large datasets)
  • Parallel analysis: compare actual eigenvalues to those from random data — more accurate than the Kaiser criterion (recommended)

Rotation method

Rotation makes factors more interpretable. Two families:

Family Method Use when
Orthogonal Varimax You expect factors to be uncorrelated
Oblique Oblimin / Promax You expect factors to correlate (common in psychology)

In psychology, oblique rotation is usually more realistic.

What loadings tell you

A factor loading is the correlation between an item and a factor. Loadings > .30 (or .40 for a more conservative threshold) are typically considered meaningful. An item should load strongly on one factor and minimally on others — check for cross-loadings.

Go to Factor → Exploratory Factor Analysis.

Confirmatory Factor Analysis (CFA)

When to use it

  • You have a theoretically specified model of which items load on which factors
  • Testing whether a previously validated scale replicates in your sample
  • Comparing alternative factor structures (e.g., one-factor vs. two-factor)
  • As a prerequisite to testing measurement invariance across groups

Model fit indices

Index Acceptable range Notes
χ² Non-significant (p > .05) Very sensitive to sample size — rarely met with large N
CFI / TLI > .90 (acceptable), > .95 (good) Comparative Fit Index / Tucker-Lewis Index
RMSEA < .08 (acceptable), < .05 (good) Root Mean Square Error of Approximation
SRMR < .08 Standardised Root Mean Square Residual
TipAlways report multiple fit indices

No single fit index is sufficient. Report CFI, TLI, RMSEA (with 90% CI), and SRMR together.

Principal Component Analysis (PCA)

When to use it

  • You want to reduce a large number of variables to a smaller set of components for use in further analyses
  • You want to remove multicollinearity by replacing correlated variables with orthogonal components
  • You are not primarily interested in a latent variable model

How to do it in JASP

Reliability: Cronbach’s α

When developing or using a scale, report Cronbach’s α as a measure of internal consistency. Values > .70 are conventionally considered acceptable.

Cronbach’s α is available in JASP under Factor → Reliability Analysis. Note that α depends on the number of items and their average inter-correlation, so it is not a pure measure of unidimensionality.

Further reading

  • DSUJ chapter on factor analysis (check your edition) — EFA and CFA in JASP with worked examples.
  • Brown, T. A. (2015). Confirmatory factor analysis for applied research (2nd ed.). Guilford Press. — The applied CFA standard; chapters 1–5 cover the EFA/CFA distinction, model specification, fit, and revision.
  • Kline, R. B. (2023). Principles and practice of structural equation modeling (5th ed.). Guilford Press. — Chapter 14 gives a compact CFA treatment in the broader SEM context.
  • For measurement equivalence across groups, see the Measurement Invariance page.

TipFor more substantial practice

A genuine multi-item scale is more informative than a four-variable demo. The Big Five Inventory (50 Likert items, freely available from the Open Psychometrics Project) is a classic worked example: import the CSV into JASP, fit an EFA with five factors and oblimin rotation, and inspect whether the resulting loadings recover the OCEAN dimensions.
